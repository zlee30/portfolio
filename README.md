# Personal Portfolio

The personal portfolio website of Zach 

**Live site:** [zlee30.github.io/portfolio](https://zlee30.github.io/portfolio)

## About

This site was designed and built from scratch to showcase my projects and give people a quick way to learn who I am and what I build. 

## Features

- Responsive layout that works on desktop, tablet, and mobile
- Smooth scroll navigation between sections
- Interactive project gallery with a full-screen image viewer
- Clean, minimal design focused on content
- Fast loading with no external frameworks

## Tech stack

- **HTML** — structure and semantic markup
- **CSS** — responsive design, transitions, and layout
- **JavaScript** — smooth scrolling and interactive gallery
- **GitHub Pages** — free hosting and deployment

## Sections

- **Home** — a quick intro with profile photo and tagline
- **Projects** — a grid of featured projects with clickable cards
- **About** — more about me, my background, and my interests
- **Contact** — ways to get in touch

## Featured projects

- **AI email responder bot** — a Python bot that scans Gmail for unread messages and generates context-aware replies using Claude
- **This portfolio** — the site you're currently reading about
- *More coming soon*

## Running locally

If you want to run the site on your own computer:

1. Clone the repository
git clone https://github.com/zlee30/portfolio.git
2. Open the folder
cd portfolio
3. Open `index.html` in your browser

That's it — no build step, no dependencies.
